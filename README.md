# Mi Agencia de Viajes - Proyecto Django

Proyecto web desarrollado con Django como entrega final del curso.

## Funcionalidades principales

- Registro de usuarios (Signup)
- Login y logout
- Edición de perfil
- Avatar de usuario
- Blog de publicaciones (CRUD completo)
- Vista en detalle de publicaciones
- Subida de imágenes
- Texto enriquecido con CKEditor
- Mensajería entre usuarios
- Bandeja de entrada
- Bandeja de enviados
- Panel de administración Django

## Tecnologías utilizadas

- Python
- Django
- SQLite
- Bootstrap
- CKEditor

## Instalación del proyecto

Clonar el repositorio:

git clone https://github.com/maxiaredes/MiPrimerPagina-Aredes.git


Entrar al proyecto:

cd MiPrimerPagina-Aredes


Crear entorno virtual:

python -m venv venv


Activar entorno virtual (Windows):

venv\Scripts\activate


Instalar dependencias:

pip install -r requirements.txt


Aplicar migraciones:

python manage.py migrate


Crear superusuario:

python manage.py createsuperuser


Ejecutar servidor:

python manage.py runserver


Abrir en navegador:

http://127.0.0.1:8000/
